# CleanupTemp Pro - Система тем

## 🎨 Добавлена поддержка светлой и тёмной темы!

### Что нового:

1. **ThemeManager.cs** - новый класс для управления темами
2. **Кнопка переключения темы** - в правом верхнем углу интерфейса
3. **Две темы:**
   - 🌙 **Тёмная тема** (по умолчанию) - для работы в тёмное время
   - ☀ **Светлая тема** - для работы днём

### Как использовать:

1. Добавьте файл `ThemeManager.cs` в ваш проект
2. Замените `CleanupForm.cs` на обновлённую версию `CleanupForm_Enhanced.cs`
3. Скомпилируйте проект
4. В приложении нажмите кнопку "🌙 Тема" в правом верхнем углу для переключения темы

### Особенности реализации:

#### ThemeManager
- Централизованное управление цветами
- Два набора цветовых схем (тёмная и светлая)
- Автоматическое применение темы ко всем элементам интерфейса

#### Цветовые схемы:

**Тёмная тема:**
- Фон: тёмно-серый (#2D2D30)
- Текст: белый и светлые оттенки
- Акценты: яркие цвета для выделения

**Светлая тема:**
- Фон: светло-серый и белый (#F0F0F0, #FFFFFF)
- Текст: тёмно-серый (#1E1E1E)
- Акценты: приглушённые цвета

### Автоматическое сохранение темы:

В будущем можно добавить сохранение выбранной темы в настройках:
```csharp
// Сохранение
Properties.Settings.Default.Theme = ThemeManager.CurrentTheme.ToString();
Properties.Settings.Default.Save();

// Загрузка
ThemeManager.CurrentTheme = (ThemeType)Enum.Parse(typeof(ThemeType), 
    Properties.Settings.Default.Theme);
```

### Примеры использования ThemeManager в коде:

```csharp
// Получить цвет фона
Color bg = ThemeManager.GetBackground();

// Получить цвет текста
Color text = ThemeManager.GetTextPrimary();

// Получить цвет кнопки
Color btnColor = ThemeManager.GetButtonPrimary();

// Применить тему ко всей форме
ThemeManager.ApplyTheme(this);

// Переключить тему
ThemeManager.CurrentTheme = ThemeType.Light;
ThemeManager.ApplyTheme(this);
```
