# CleanupTemp Pro 🧹

Программа для очистки временных файлов Windows с графическим интерфейсом.

## Описание

CleanupTemp Pro — это утилита для Windows, которая помогает освободить место на диске, удаляя временные файлы и кэш системы.

## Возможности

- 🗑️ Очистка временных файлов Windows
- 📊 Графический интерфейс для удобной работы
- ⚡ Быстрая и безопасная очистка
- 🎨 Современный дизайн интерфейса

## Системные требования

- Windows 10/11
- .NET 8.0 или выше

## Установка

1. Скачайте последнюю версию из [Releases]([https://github.com/vitalikkontr/CleanupTemp-Pro/releases])
2. Запустите установщик
3. Следуйте инструкциям мастера установки

## Использование

1. Запустите программу
2. Выберите категории файлов для очистки
3. Нажмите кнопку "Очистить"
4. Дождитесь завершения процесса

## Разработка

### Технологии

- C#
- .NET 8.0
- Windows Forms

### Сборка проекта

```bash
git clone https://github.com/vitalikkontr/CleanupTemp-Pro.git
cd CleanupTemp-Pro
dotnet build
```
