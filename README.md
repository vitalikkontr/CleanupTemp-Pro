# CleanupTemp Профессиональная 🧹

Программа для очистки временных файлов Windows с графическим интерфейсом.

## Описание

CleanupTemp — это утилита для Windows, которая помогает освободить место на диске, удаляя временные файлы и кэш системы.

## Возможности

🗑️ Очистка временных файлов Windows


📊 Удобный графический интерфейс

💡 Подсказки ToolTip для кнопок

⚡ Быстрая и безопасная очистка системы

🎨 Современный и аккуратный дизайн интерфейса + две темы черная и белая

## Системные требования

- Windows 10/11
- .NET 8.0 или выше

## 🔒 Безопасность

- Программа работает только с системными временными папками
- Не удаляет важные системные файлы
- Требует прав администратора для безопасной очистки
- Все операции логируются

## 📋 Что очищается?

- `%TEMP%` - временные файлы пользователя
- `C:\Windows\Temp` - системные временные файлы
- Корзина Windows
- Кэш браузеров (опционально)
- Prefetch файлы

## 📥 Установка

1. [Скачайте последнюю версию](https://github.com/vitalikkontr/CleanupTemp-Pro/releases/latest)
2. Запустите установщик CleanupTemp-Professional-Setup-v4.2.exe
3. Следуйте инструкциям мастера установки

## Использование

1. Запустите программу **от имени администратора** (правой кнопкой → Запуск от имени администратора)
2. Выберите категории файлов для очистки
3. Нажмите кнопку "Очистить"
4. Дождитесь завершения процесса
